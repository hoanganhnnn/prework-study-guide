HTML

  • HTML - gives the content structure.
  • The head element contains information about the webpage. Elements included within the head element are not visable when a user looks at a webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for a webpage, this is required and is extremely important for SEO (search engine optimization).
  • The base element connects an external reource to the HTML document.
  • The script element embeds client-side scripts in the HTML document.
  • There are different levels of heading to organize a page, starting with h1, h2, etc.
  • ul, ol, and li are elements represent unordered lists, ordered lists, and list items in HTML.
  • The img element can contain various attributes; the src attribute is most important because it defines the location of the image.

CSS

  • CSS - gives the content style.
  • There are three types of CSS: inline CSS, internal style sheet, and external CSS style sheet. Te external CSS style sheet is most preferred as it allows developers to keep all CSS rules in a separate file, making design changes easier.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain 2 important components: the CSS property that is wanting to be applied and the value of the property.
  • There are 2 things you have to assign CSS to: class attribute as the selector and CSS rule based on the element name.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: update the local branch
  • git push origin feature/"yourbranchhere": push up changes to remote repository

JavaScript

  • JavaScript - handles logic and allows users to interact with a website.
  • A variable is named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is like a decision making in the code, if state meets a condition, the code will be executed.
  • An array is a single variable that is used to hold a group of data that is related in some way.
  • The for loop uses predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • Functions is a set of instructions to tell the computer to perform a certain task.
  • There are 2 requirements for a JS function: define the function (ex: function declaration) and call the function (nameoffunction();).
